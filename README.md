# Sistema de Leilões

## Sobre o projeto

Sistema desktop desenvolvido em Java para gerenciamento de uma casa de leilões. A aplicação permite cadastrar produtos para leilão, listar todos os itens disponíveis e registrar vendas, além de exibir um relatório dos produtos já vendidos.

## Funcionalidades

- Cadastro de produtos com nome e valor
- Listagem de todos os produtos cadastrados
- Registro de venda de produtos
- Consulta de produtos vendidos

## Tecnologias utilizadas

- Java (Swing)
- MySQL
- NetBeans IDE
- JDBC (mysql-connector-java)
